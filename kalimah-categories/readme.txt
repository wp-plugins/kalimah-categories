Improved interface for categories display

== Description ==
This plugin displayed an improved user interface for categories for a better and easier categories management.

== Installation ==
1. Upload plguin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the plugin through plugin managment in Wordpress dashboard under Posts scetions
